1. take clone.

2. setup frontend 
=> cd frontend
=> npm i 
=> npm start => check web page => close
=> npm run build -> creates build

3. setup dunjo app
=> python3 manage.py makemigrations
=> python3 manage.py makemigrations
=> python3 manage.py migrate
=> python3 manage.py createsuperuser   

4. enable dev mode from setting.py

5. create a static folder on root.

6. python3 manage.py collectstatic

7. run the app => python3 manage.py runserver
